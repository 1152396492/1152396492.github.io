总结

由于懒得想所以就借鉴了题解中的一片代码改了一下；
他的关键代码的else if中，他的为
low[u] = min ( low[u] , low[v] ) ;
在此题中是正确的，但按照标准的写法应该为如下；
其中的关键代码为：
if(!dfn[v])Tarjan(v) , low[u] = min( low[u], low[v] ) ;
else if (!col[v]) low[u] = min (low[u] , dfn[v] ) ;
题解的写法在求强连通分量时是等价的，但求割点时是错误的，所以严谨来说应该写为
low[u] = min (low[u] , dfn[v] ) ;
